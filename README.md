# 上級英単語 100 ｜ 大学入試 上級語彙 学習Webアプリ

> 私大入試・共通テストの読解で狙われる **上級寄りの英単語100語** を、スマホで美しく学べる学習アプリです。
> 原典「大学入試 上級英単語 100」の全100語（意味＋ポイント）を完全収録し、例文・関連語マップを独自に付与しました。
>
> **© 2026 上級英単語100 — Created on June 5, 2026（2026年6月5日 制作）**

---

## 1. プロジェクトの目的・コンセプト

- **目的**：上級英単語100語を「読む・覚える・確認する」までスマホ1台で完結させる。
- **デザイン**：紙とインクを思わせる **和モダン × ミニマル**。落ち着いた藍緑・朱・古金・苔色の配色。
- **動き**：`cubic-bezier` による **ぬるぬると滑らかなアニメーション**（フリップ・スワイプ・スライド）。
- **制約の遵守**：
  - ❌ **ネオン色調は不使用**（落ち着いた自然色のみ）
  - ❌ **進捗バーは不使用**（代わりに「リング」「ステップドット」「数値」で進み具合を表現）
- **著作情報**：設定画面・フッタに「2026年6月5日 制作 / © 2026」を明記。

## 2. 完成済みの機能

| 機能 | 概要 |
|------|------|
| 🏠 ホーム | ヒーロー、今日の1語、習得度リング、クイック起動、覚え方のコツ、概要 |

| 🎲 気まぐれ1語 | ヘッダのサイコロからランダム1語を表示 |
| 🌗 テーマ | ライト（生成り紙）／ダーク（墨夜）切替、`prefers-reduced-motion` 尊重 |

## 3. 機能エントリ（パス・パラメータ）

本アプリは **SPA（単一HTML）** です。ビューはタブ／クイックカードで切り替わります（URLパラメータは使用しません）。

| 入口 | 操作 | 表示ビュー |
|------|------|-----------|
| `index.html` | 初期表示 | ホーム |
| タブバー | ホーム / 一覧 / カード / クイズ / 設定 | 各ビュー |
| ホームのクイックカード | はじめる | カード／クイズ／一覧／お気に入り／**つづりテスト** |
| 単語カード タップ | — | 詳細ボトムシート |
| ヘッダ 🎲 | — | ランダム1語の詳細シート |

> データ永続化は **ブラウザの localStorage**（キー: `advWords100.v1`）。外部APIやサーバDBは使用していません（静的サイト）。

## 5. データモデル

### 単語（WORD_DATA）
```js
{
  no, word, pos, posKey,   // posKey: verb|adj|noun|mixed
  initial,                 // 索引用 先頭アルファベット
  meaning, point,          // 意味・ポイント（HTML可）
  tags: []                 // 多義語/派生重要/対義語ペア/発音注意/経済/社会 ...
}
```

### 例文（EXAMPLES） … `{ [no]: { en, ja } }`
### 関連語（RELATIONS） … `{ [no]: { deriv, syn, anto, collo, confuse } }`

### 保存データ（localStorage: `advWords100.v1`）
```js
{
  learned: [no...],        // 習得済み
  fav: [no...],            // お気に入り
  settings: { theme, autoSpeak },
  stats: { flashRounds, quizRuns, bestQuiz, lastSeen }
}
```
